# Photonic Logic System  
**Early Concepts for Optical Spatial Routing + Weighted Summation with Neural-Style Activation**

Exploratory notes and diagrams for a **photonic logic / neuromorphic computing** approach using engineered lenses, spatial light routing, and analog optical summation.

## Overview

This repository documents initial brainstorming for an all-optical (or hybrid photonic) system that performs logic operations and basic neural-style computation using light instead of electrons.

Key elements explored:
- Spatial routing and mapping of binary data via engineered lenses or free-space optics ("MAP" transformation)
- Weighted optical signal summation (visualised as coloured interference/summation diagram)
- Activation threshold mechanism (e.g., activation ~5, max signal weight = 9)
- 3D photonic structures (cube/lattice sketches possibly representing waveguides, resonators, or crystal-like routing)
- Propagation notes ("rise ~ clockwise")

The goal is energy-efficient, parallel optical processing that could address limitations in traditional electronic computing for certain AI/neural workloads.

**Current stage**: Purely conceptual — handwritten notebook sketches (April 2026). No working hardware or full simulation yet.

## Motivation

Traditional electronic logic and neural networks face power and speed walls at scale. Photonics offers potential advantages through massive parallelism, low-loss propagation, and natural interference-based operations. This project explores a spatial/lens-based approach to encoding, routing, and activating signals optically.
